# G-Maps-Scraper-v4.1
G-Maps (Google Maps) Scraper v4.1 — Developed a most powerful tool g-maps data scraper v4.1 to Extract leads, emails, phone numbers &amp; more from google maps places. This bot can easily scrape thousands of leads within a few hours. It's very useful for business outreach, lead generation, cold emailing, and market research.

# Key features:
- Get searching keywords and locations list from the user
- Search one keyword in first location and so on, and then second keyword in all locations
- Extract all data from google maps
- Find emails from their websites, sub-pages, and social media pages
- Export the data on completion automatically, user can also export anytime manually
- Saves all time data into the SQLite3 Database
- Notify user on email about the progress of scraping if user is far away from the pc
- Notify user on email when it gets completed
- Auto updates system
